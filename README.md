Interactive ISA Simulator with Fetch-Decode-Execute Cycle VisualizationAn interactive, web-based educational tool designed to visualize the internal operations of a Central Processing Unit (CPU).  Computer Architecture concepts such as instruction execution, register manipulation, memory operations, and the Fetch-Decode-Execute cycle are notoriously difficult for students to understand because these processes occur internally inside the CPU and cannot be physically observed. This project solves that problem by replacing static diagrams with a dynamic, step-by-step execution engine.  Key FeaturesReal-Time FDE Cycle Visualization: Watch instructions move step-by-step through the Fetch, Decode, and Execute stages.  Interactive Assembly Compiler: Allows users to enter assembly-like instructions and immediately observe the CPU processing them.  Dynamic Register & Memory Tracking: Visually demonstrates register and memory operations with real-time updates.  Execution Logs: Integrated event logging to translate raw microcode operations into a readable execution history.  Advanced UI Animations: Features highly tactile visual cues like read/write highlighting, data bus transfer animations, and instruction decoding scrambles to make learning engaging.  Tools and TechnologiesThe project is built using a modern, modular frontend architecture.  TechnologyPurposeReactFrontend development and UI components. Supported Instruction Set Architecture (ISA)The simulator focuses on a simplified set of core instructions to demonstrate fundamental data flow:  LOAD Rd, Addr – Loads data from a memory address into a register.  STORE Rs, Addr – Stores data from a register into a memory address.  ADD Rd, Rs – Performs an Arithmetic Logic Unit (ALU) addition.  SUB Rd, Rs – Performs an Arithmetic Logic Unit (ALU) subtraction.
